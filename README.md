# Social Network Challenge
***


## REST API:
```
/signup                                      POST
/login                                       POST

/profile                                     GET
/profile/posts                               GET
/profile/posts/{postId}/comments             GET

/posts                                       GET, POST
/posts/{postId}                              GET, PUT
/posts/{postId}/comments                     GET, POST
/posts/{postId}/comments/{commentId}         GET, PUT
```
